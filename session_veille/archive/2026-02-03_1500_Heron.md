# Veille - Heron (Heron Data)

**Date** : 2026-02-03
**Sujet** : Heron Data (Fintech/Insurtech - Underwriting Automation)
**Source** : Internal Knowledge (YC S20)

## Résumé
Heron Data est une infrastructure fintech utilisée dans les processus de souscription (lending/assurance) pour automatiser l'analyse des données financières brutes (relevés bancaires, PDF).

## Points Clés
- **Fonctionnalité** :
    - **Document Extraction** : OCR haute fidélité pour relevés bancaires et financiers.
    - **Categorization** : Labelisation automatique des transactions (revenus, paie, dettes).
    - **Underwriting Metrics** : Calcul automatique de scores, cash-flow, MRR, DSCR.
    - **Détection Fraude** : Vérification de la manipulation de documents.
- **Cible** : Prêteurs SME, Fintechs, Assureurs (pour l'analyse de risques financiers).
- **Technologie** : API-first, Machine Learning pour la catégorisation.
