# Veille - DeepOpinion (Otera)

**Date** : 2026-02-03
**Sujet** : DeepOpinion / Otera (Insurtech - AI Automation / Claims)
**Source** : Web Search (DeepOpinion.ai, Tech.eu)

## Résumé
DeepOpinion (rebranding en **Otera**) est une plateforme d'automatisation par "Agentic AI" spécialisée dans le traitement des données non structurées pour l'assurance (Sinistres et Souscription).

## Points Clés
- **Concept** : "Dark Processing" (traitement sans intervention humaine).
- **Technologie** :
    - **Agentic AI** : Agents autonomes capables de raisonner sur des workflows complexes.
    - **Intelligent Document Processing (IDP)** : Traitement de 35k+ layouts avec 98.3% de précision.
    - **Zero-Shot Learning** : Modèles pré-entraînés sur le domaine assurance.
- **Cas d'usage Sinistres** :
    - Classification automatique des emails/documents.
    - Extraction de données sur rapports médicaux/experts.
    - Straight-Through Processing (STP) > 65%.
- **Financement** : 11M€ Series A (Sept 2024).
- **Clients** : Allianz, Uelzener, etc.
