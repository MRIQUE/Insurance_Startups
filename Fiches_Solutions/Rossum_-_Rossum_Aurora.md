## Description
Rossum est une plateforme cloud-native d'Intelligent Document Processing (IDP) propulsee par un LLM transactionnel proprietaire (T-LLM) appele Aurora, entraine sur plus de 11 millions de documents transactionnels. Contrairement aux LLM generatifs classiques, Aurora utilise un decodeur discriminatif qui ne genere jamais de texte nouveau : il extrait uniquement les donnees presentes dans le document, eliminant ainsi tout risque d'hallucination. La plateforme automatise l'ensemble du cycle de traitement documentaire B2B — de l'ingestion multicanal (email, scanner, PEPPOL, API) a la validation croisee avec les ERP et systemes metier, en passant par l'extraction de donnees, le rapprochement et le routage d'approbation.

**Proposition de valeur** : Automatisation de bout en bout du traitement des documents transactionnels (factures, bons de commande, declarations douanieres, formulaires d'assurance) avec une precision superieure a 92,5% des le premier jour, sans template ni configuration par type de document, dans 276 langues incluant l'ecriture manuscrite.

### Le probleme adresse
Les entreprises traitent des volumes massifs de documents transactionnels (factures, bons de commande, contrats, formulaires de sinistres) de maniere largement manuelle. Ce traitement manuel est lent, couteux, sujet aux erreurs et ne passe pas a l'echelle. Les solutions OCR traditionnelles necessitent la creation et la maintenance de templates par type de document et par fournisseur, ce qui limite leur adaptabilite. Rossum resout ce probleme en offrant une extraction IA sans template, capable de comprendre le contenu et la structure de n'importe quel document transactionnel sans configuration prealable.

- **Template-free** : aucune configuration par type de document ou fournisseur
- **276 langues** supportees, incluant l'ecriture manuscrite
- **Architecture cloud-native** : microservices, extensions low-code, API REST
- **Conformite** : ISO 27001, SOC 2, HIPAA, RGPD
- **Detection de fraude** alimentee par IA
- **Aurora 1.5** (2024) : traitement de documents de 100+ pages, 4x plus rapide
